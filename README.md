# Java Developer Internship – Task 9

## Task
Collections Framework – Student Management System

---

## Objective
The objective of this task is to understand and apply the Java Collections Framework by building a student management system. 
The program uses List, Set, Map, and Comparator to store, sort, remove duplicates, and efficiently retrieve student data.

---

- IntelliJ IDEA

## Program Description
- Stores student objects in ArrayList
- Removes duplicate students using HashSet
- Uses HashMap for fast student lookup
- Sorts students using Comparator
- Iterates collections using enhanced for loop
- Prints formatted student reports
